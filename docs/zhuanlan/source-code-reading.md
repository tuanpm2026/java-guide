## Giới thiệu

**《Java Must-read Source Code Series》** là một e-book nội bộ trong [Knowledge Planet](../about-the-author/zhishixingqiu-two-years.md) của tôi. Hiện đã tổng hợp source code của các framework/middleware như Dubbo 2.6.x, Netty 4.x, Spring Boot 2.1. Sau này sẽ tổng hợp thêm nhiều source code chất lượng đáng đọc, đang hoàn thiện liên tục.

Cấu trúc rõ ràng, nội dung chi tiết, rất phù hợp để đọc cho những bạn muốn học sâu về source code của framework/middleware.
