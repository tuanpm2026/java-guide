Nội dung phần này là độc quyền của [Knowledge Planet](../about-the-author/zhishixingqiu-two-years.md) của tôi. Hiện đã cập nhật các chuyên mục sau:

- [《Java Interview Guide》](./java-mian-shi-zhi-bei.md): Bổ sung nội dung cho phiên bản open source của JavaGuide!
- [⭐AI Smart Interview Assistant Platform + RAG Knowledge Base](./interview-guide.md): Phát triển dựa trên Spring Boot 4.0 + Java 21 + Spring AI 2.0. Rất phù hợp làm project học tập và resume, ngưỡng học tập thấp, giúp nâng cao khả năng cạnh tranh trong tuyển dụng, là project thực chiến tập trung vào việc làm.
- [《Backend Interview High-frequency System Design & Scenario Questions》](./back-end-interview-high-frequency-system-design-and-scenario-questions.md): Bao gồm các case study system design phổ biến như short URL system, flash sale system và các scenario questions tần suất cao như deduplication of massive data, third-party authorization login.
- [《Handwritten RPC Framework》](./handwritten-rpc-framework.md): Triển khai một RPC framework đơn giản từ đầu dựa trên Netty + Kryo + Zookeeper.
- [《Java Must-read Source Code Series》](./source-code-reading.md): Hiện đã tổng hợp source code của Dubbo 2.6.x, Netty 4.x, Spring Boot 2.1 và các framework/middleware khác.
- ……

Hoan nghênh các bạn chuẩn bị phỏng vấn Java và học Java tham gia [Knowledge Planet](../about-the-author/zhishixingqiu-two-years.md) của tôi — có rất nhiều nội dung chất lượng! Dù phí rất phải chăng như rau, nhưng nội dung trong Planet còn chất lượng hơn nhiều so với các khóa học training hàng chục triệu.

Tôi có nguyên tắc riêng của mình — không "cắt cỏ", làm nội dung nghiêm túc, thực lòng muốn giúp ích cho bạn!
