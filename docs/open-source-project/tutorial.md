## Java

- [JavaGuide](https://github.com/Snailclimb/JavaGuide "JavaGuide"): 【Java Learning + Interview Guide】 Một bộ tài liệu bao gồm hầu hết kiến thức cốt lõi mà Java programmer cần nắm.
- [toBeBetterJavaer](https://github.com/itwanger/toBeBetterJavaer): Hướng dẫn học Java dễ hiểu và hài hước, nội dung bao gồm Java basics, Java Collections Framework, Java Concurrent Programming, JVM, Java Enterprise Development (Git, SSM, Spring Boot) và các kiến thức khác.
- [interview-guide](https://github.com/csguide-dabai/interview-guide): Tổng hợp các trọng tâm trong backend interview questions, hy vọng có thể giúp ích cho các bạn chuẩn bị phỏng vấn campus recruitment cho vị trí internet development.
- [advanced-java](https://github.com/doocs/advanced-java "advanced-java"): Kiến thức nâng cao dành cho Java engineer internet: bao gồm kiến thức trong các lĩnh vực high concurrency, distributed, high availability, microservices, massive data processing.
- [toBeTopJavaer](https://github.com/hollischuang/toBeTopJavaer "toBeTopJavaer"): Con đường trở thành Java engineer xuất sắc.
- [technology-talk](https://github.com/aalansehaiyang/technology-talk): Tổng hợp các technology framework phổ biến trong Java ecosystem, open source middleware, system architecture, database, big company architecture cases, common third-party libraries, project management, online issue troubleshooting, personal growth và các kiến thức khác.
- [JCSprout](https://github.com/crossoverJie/JCSprout): Java core knowledge base ở giai đoạn đang phát triển.
- [bestJavaer](https://github.com/crisxuan/bestJavaer): Đây là một series tutorial để trở thành Java programmer tốt hơn.
- [java-design-patterns](https://github.com/iluwatar/java-design-patterns "java-design-patterns"): Design patterns được triển khai bằng Java.

## Computer Basics

- [cs-self-learning](https://github.com/PKUFlyingPig/cs-self-learning): Computer self-study guide, tổng hợp các khóa học máy tính chất lượng cao của nhiều trường đại học danh tiếng Âu Mỹ.
- [CS-Notes](https://github.com/CyC2018/CS-Notes "CS-Notes"): Kiến thức cơ bản bắt buộc cho technical interview, Leetcode solutions, backend interview, Java interview, spring/fall recruitment, operating system, computer network, system design.
- [Waking-Up](https://github.com/wolverinn/Waking-Up): Tổng hợp toàn diện câu hỏi phỏng vấn về computer basics (computer network/operating system/database/Git...).

## System Design

### SpringBoot

- [springboot-guide](https://github.com/Snailclimb/springboot-guide): Tổng hợp kiến thức cốt lõi SpringBoot. Dựa trên Spring Boot 2.19+.
- [SpringAll](https://github.com/wuyouzhuguli/SpringAll "SpringAll"): Học từng bước, học Spring Boot, Spring Boot & Shiro, Spring Cloud, Spring Security & Spring Security OAuth2, blog Spring series source code.
- [Springboot-Notebook](https://github.com/chengxy-nds/Springboot-Notebook): Series case study tổng hợp sử dụng Spring Boot làm development framework cơ bản, kết hợp Redis, Rabbitmq, ES, MongoDB, Spring Cloud, Kafka, Skywalking và các mainstream internet technologies khác, triển khai các function points phổ biến.
- [springboot-learning-example](https://github.com/JeffLi1993/springboot-learning-example "springboot-learning-example"): Spring Boot practical learning cases, best practice cho Spring Boot beginners và consolidation của core technologies.
- [spring-boot-demo](https://github.com/xkcoding/spring-boot-demo "spring-boot-demo"): spring boot demo là một project để học sâu và thực hành spring boot, hiện có tổng cộng 63 integration demo, đã hoàn thành 52 cái.
- [SpringBoot-Labs](https://github.com/YunaiV/SpringBoot-Labs): Spring Boot series tutorial.

### Nginx

- [nginx-tutorial](https://github.com/dunwu/nginx-tutorial): Series Nginx minimal tutorials bao gồm các nội dung thực chiến như HTTP reverse proxy, HTTPS reverse proxy, load balancing, static site, file server setup.

## Big Data

- [juicy-bigdata](https://github.com/datawhalechina/juicy-bigdata): Juicy Big Data — introductory course về các nội dung liên quan đến big data technologies.
- [flink-learning](https://github.com/zhisheng17/flink-learning "flink-learning"): Bao gồm Flink introduction, concepts, principles, practice, performance tuning, source code analysis và các nội dung khác.

## Open Source Books

- [《The Philosophical Principles of High Concurrency》](https://github.com/johnlui/PPHC): Mục tiêu của cuốn sách là trong phạm vi nhận thức hạn chế của tác giả, thảo luận về một nguyên lý triết học ẩn sau vấn đề high concurrency — tìm ra single point, tiến hành splitting.
- [《Effective Java (3rd Edition)》 Chinese-English bilingual](https://github.com/clxering/Effective-Java-3rd-edition-Chinese-English-bilingual): Tài liệu học tập Chinese-English cho các chương của 《Effective Java (3rd Edition)》.
- [《DDIA (Designing Data-Intensive Applications)》 Chinese version](https://github.com/Vonng/ddia): Chinese translation of 《Designing Data-Intensive Application》DDIA.
- [《Phoenix Architecture》](https://github.com/fenixsoft/awesome-fenix): Thảo luận về cách xây dựng một large-scale distributed system đáng tin cậy.
- [《Patterns of Distributed Systems》 Chinese version](https://github.com/dreamhead/patterns-of-distributed-systems): Chinese translation of 《Patterns of Distributed Systems》.
